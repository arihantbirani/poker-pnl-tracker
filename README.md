This project is a full-stack session analytics dashboard designed to track and analyze structured financial activity over time.
It supports secure user authentication, real-time data updates, and derived performance metrics such as hourly rates and cumulative trends.
Built using Firebase (Authentication + Firestore) and a JavaScript frontend, the system emphasizes data modeling, access control, and interactive visualization.


Check out the live tracker here: https://poker-pnl-tracker.vercel.app/
